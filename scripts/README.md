Utilitários de desenvolvimento e operação.

## Scripts principais

| Script | Descrição |
|--------|-----------|
| `configurar-fcm-v1.ps1` | Configura chave FCM v1 para push |
| `copiar-google-services.ps1` | Copia google-services.json para o app |
| `fazer-fcm-v1.ps1` | Fluxo completo FCM v1 |
| `validar-fcm-push.ps1` | Valida configuração de push |
| `liberar-chave-servico.ps1` | Libera chave de serviço no Firebase |
| `download-pdf-para-teste-iti.ps1` | Baixa PDF para testar verificação ITI |
| `test-iti-verify.ps1` | Testa fluxo de verificação ITI |
| `seedPrescription.ts` | Seed de receita (npm run seed) |
